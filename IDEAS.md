## Концепция продукта

> "Раз в неделю получаешь список вакансий под твой профиль — и одной кнопкой генерируешь готовый Bewerbung под любую из них."

Не платформа. Не агентство. **Инструмент**, который экономит человеку 10+ часов в неделю при поиске работы в Германии.

---

## Целевая аудитория

### Первичная (старт):
- Русскоязычные мигранты в Германии (~3.5 млн человек)
- Украинцы с временной защитой (§24)
- Русскоязычные с Blue Card, рабочими визами
- Недавно приехавшие (2022+) — острая боль прямо сейчас

### Вторичная (расширение с месяца 7+):
- Немецкоязычный рынок — все соискатели в Германии
- Двуязычный интерфейс (RU + DE)

---

## Боли пользователей (подтверждены исследованиями)

1. **Зарплата не указана** — 75% хотят видеть зарплату ДО подачи заявки, 59% не подают заявку без цифр
2. **Повторный ввод данных** — каждый сайт требует заново заполнять резюме
3. **Спам и нерелевантные вакансии** — StepStone/Indeed игнорируют фильтры
4. **Ghosting** — 54% не получили ответа, 69% заghosted после собеседования
5. **Медленный интерфейс** — StepStone грузится медленно, перегружен
6. **Нет единого профиля** — нужно 5 аккаунтов на 5 сайтах
7. **Сложный процесс Bewerbung** — Anschreiben + Lebenslauf занимает часы
8. **Fake-вакансии** — Indeed и StepStone полны мошеннических объявлений
9. **ATS-фильтры** — резюме не проходит автоматическую систему отбора работодателя

## Источники данных (API)

### Зелёные — бесплатно и легально:
- **Bundesagentur für Arbeit REST API** — `https://rest.arbeitsagentur.de/jobboerse/jobsuche-service/pc/v4/jobs`
  - Ключ: `X-API-KEY: jobboerse-jobsuche`
  - Поддерживает: регион, радиус, тип занятости, профессия
  - Протестировано: 201 вакансия по запросу "LKW Fahrer Köln 25km"
- **ArbeitNow API** — бесплатный, прямо из ATS работодателей, часто есть зарплата
- **Jooble Affiliate API** — агрегатор, бесплатно + платит CPC за трафик

### Красные — не использовать:
- StepStone — нет API, ToS запрещает scraping
- Indeed — Publisher API закрыт, ToS запрещает
- Google Jobs / SerpApi — судебный иск от Google (декабрь 2025)
- XING — нет публичного API

## Функциональность продукта

### Фаза 1 — Агрегатор (недели 1–5)

#### Регистрация и профиль пользователя:
- Профессия / навыки (текст + теги)
- Город и радиус поиска (км)
- Тип занятости: Vollzeit / Teilzeit / Minijob / Freelance
- Язык вакансий: немецкий / английский / оба
- Загрузка текущего резюме (PDF или текст)

#### Агрегация вакансий:
- Сбор с трёх источников: Bundesagentur + ArbeitNow + Jooble
- Дедупликация: одна вакансия с разных сайтов = одна запись
- Фильтрация под профиль пользователя (AI-матчинг)
- Кэширование: хранить что уже показывали, показывать только новое

#### Список вакансий (интерфейс):
- Карточки: название, компания, город, расстояние, тип занятости, дата
- Зарплата — если указана (с 2026 всё больше компаний обязаны указывать)
- Фильтры на странице
- Кнопка "Подготовить Bewerbung" на каждой карточке

#### Уведомления:
- Еженедельный дайджест в Telegram (пятница)
- Только новые вакансии — без дублей предыдущих недель
- Ссылка на сайт для просмотра полного списка

#### Фильтр fake-вакансий (AI-проверка):
Вакансия помечается как подозрительная если:
- Нет реального адреса компании
- Запрашивают личные данные (паспорт, банк) сразу в форме
- Зарплата нереально высокая для профессии
- Компания не существует в Handelsregister
- Орфографические ошибки в описании
- Контакт только через WhatsApp/Telegram (не корпоративный email)
- Недавно созданный домен работодателя (< 6 месяцев)

---

### Фаза 2 — Bewerbung (недели 6–16)

#### Генерация Anschreiben:
- AI генерирует на основе: вакансии + профиля пользователя
- Правильная немецкая структура: Datum, Betreff, Anrede, Hauptteil, Schluss
- Адаптирован под конкретную вакансию и компанию
- Многоступенчатая проверка качества:
  1. Структурная проверка (блоки, длина, формат)
  2. Языковая проверка (LanguageTool API — бесплатный)
  3. AI-оценка тона и релевантности (второй запрос: "оцени по 5 критериям")
  4. Финальный документ → PDF

#### Адаптация Lebenslauf:
- Берёт загруженное резюме пользователя
- Адаптирует под требования конкретной вакансии
- **ATS-оптимизация**: ключевые слова из вакансии вшиваются органично
- Правильный немецкий формат (не российский)
- Машиночитаемый PDF (не картинка) — проходит ATS-системы
- НЕ использовать таблицы, колонки, изображения (ATS их не читает)
- Стандартные шрифты (Arial, Calibri, Times New Roman)

#### Доставка пользователю:
- PDF Anschreiben — скачать
- PDF Lebenslauf — скачать
- Email HR компании (найденный автоматически)
- Ссылка на форму компании (если есть)
- Кнопка "Скопировать текст письма"
- **Пользователь отправляет сам** — мы не отправляем за него
  (причина: юридическая ответственность + ненадёжность автоотправки)

### Монетизация

#### Freemium модель:
- **Бесплатно всегда:** поиск и агрегация вакансий, еженедельный дайджест

- **Платно разово:** €4.99 за один пакет (Anschreiben + Lebenslauf)

## Конкурентный анализ

### Прямые конкуренты:
- **Lebenslauf.de** — только резюме, 1.4 млн пользователей, нет агрегации вакансий
- **HeyJobs.co** — агрегатор, нет генерации документов
- **Die-Bewerbungsschreiber.de** — ручное написание, €100+, 1-4 дня ожидания

### Ключевое отличие BewerbZen:
Никто не делает **связку агрегация вакансий + AI-генерация документов** в одном месте.

### Почему не конкурируем с ChatGPT:
- ChatGPT не знает вашу ситуацию (нет памяти профиля)
- ChatGPT не присылает еженедельный дайджест
- ChatGPT не генерирует ATS-оптимизированный PDF
- ChatGPT не находит email HR компании

## Правовые аспекты

### GDPR:
- Галочка "согласен на обработку данных" при регистрации
- Политика конфиденциальности (шаблон + адаптация)
- Данные резюме хранятся зашифрованно
- Право на удаление данных

### Позиционирование (не кадровое агентство):
- "Не ищем работу за вас. Помогаем тратить на поиск 30 минут в неделю вместо 10 часов."
- Дисклеймер: "Сгенерированные документы — шаблоны для редактирования. Не является юридической консультацией."
- Ответственность за отправку — на пользователе

### Регистрация бизнеса:
- Старт: PayPal.me / Ko-fi без юрлица (тест гипотезы)
- После первых 10 платящих: Kleingewerbe (€20-30, 1 день)
- ВАЖНО: перед стартом консультация в Migrationsberatung (§25 + Bürgergeld)
