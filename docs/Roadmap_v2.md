## Главная гипотеза

> Русскоязычные жители Германии, которые ищут работу и не получают ответов,
> готовы попробовать инструмент, который объясняет немецкие стандарты и помогает
> собрать Bewerbung за 10 минут — если они доверяют отправителю и видят конкретный результат.

**Сегмент:** русскоязычные в Германии, ищут работу 1–3 месяца, 0–5 ответов на отклики.
**Боль:** "я отправляю, но не понимаю почему нет ответов".
**Решение:** структурированный Bewerbung по немецкому стандарту + объяснение почему это работает.

## Неделя 2 (20–27 апреля) — Ручной MVP и первые интервью

### Цель
Получить 10–30 заявок. Лично помочь первым 5 людям собрать Bewerbung.

### Задачи

**Ручной MVP (Wizard of Oz):**
- [ ] Человек заполняет форму на Tally
- [ ] Ты получаешь уведомление в Telegram (через Tally webhook → n8n)
- [ ] Вручную пишешь ему в Telegram / email: "Привет, это Михаил из BewerbZen"
- [ ] Запрашиваешь: резюме (PDF или текст) + описание вакансии
- [ ] Через Claude API (или вручную) генерируешь Anschreiben (сопроводительное письмо)
- [ ] Отправляешь PDF / Google Doc с готовым Bewerbung
- [ ] Просишь обратную связь через 3–5 дней

**Шаблон Bewerbung для ручного MVP:**
- Deckblatt (обложка)
- Anschreiben (сопроводительное письмо) — под конкретную вакансию
- Lebenslauf (резюме) — по немецкому стандарту
- Документы (опционально)

**Интервью (5–10 человек):**
Вопросы:
1. Как долго ищешь работу? Сколько откликов отправил?
2. Что сейчас самое сложное?
3. Как ты сейчас делаешь Bewerbung? Сколько времени уходит?
4. Если бы был инструмент, который делает это за 10 минут — что для тебя важнее: скорость, качество или объяснения?
5. За что ты был бы готов заплатить?

## Неделя 4–5 (4–18 мая) — Автоматизация первого потока

### Telegram-бот (n8n + Claude API)

**Флоу:**
1. Пользователь пишет боту /start
2. Бот спрашивает специальность и вакансию
3. Пользователь присылает текст вакансии (или ссылку)
4. Бот просит краткое резюме (3–5 пунктов опыта)
5. Claude API генерирует Anschreiben на немецком
6. Бот отправляет текст + предлагает PDF (Gotenberg)

**n8n workflow:**
- Telegram Trigger → Parse input → Claude API → Format text → Telegram Reply
- Для PDF: Gotenberg (self-hosted или API) → File output

**Первые 3 Bewerbung — бесплатно**
**После 3-х:** Stripe Payment Link → оплата €4.99 → разблокировка

- [ ] Консультация: §25 + Bürgergeld + Kleingewerbe совместимость

## Принципы, которые НЕ меняем

1. **Автоотправку НЕ делаем** — пользователь отправляет Bewerbung сам
2. **Мы инструмент, не кадровое агентство** — важно для GDPR и лицензий
3. **Freemium** — агрегация бесплатно, Bewerbung платный
4. **Начало с русскоязычных** — потом немецкий рынок
5. **Ручной MVP сначала** — автоматизация только после сигнала спроса
