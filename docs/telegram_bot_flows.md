# BewerbZen Telegram Bot — Полная схема диалогов

## Архитектура состояний

```
IDLE ──/start──────────────► WELCOME
     ──/bewerben────────────► AWAITING_VACANCY
     ──/help────────────────► HELP
     ──/status──────────────► STATUS
     ──/jobs────────────────► JOBS_LIST
     ──/feedback────────────► AWAITING_FEEDBACK
     ──[любой текст]────────► GENERATE (MVP: любой текст = вакансия)
```

## Inline keyboard buttons (Telegram)

Все кнопки реализуются через `reply_markup.inline_keyboard`:

```json
{
  "inline_keyboard": [
    [
      {"text": "📄 Составить Bewerbung", "callback_data": "cmd_bewerben"},
      {"text": "❓ Справка",             "callback_data": "cmd_help"}
    ]
  ]
}
```

В n8n: узел Telegram → Message → включить `Reply Markup` → тип `Inline Keyboard`

## Команды для BotFather

После создания бота, настроить через /setcommands:

```
bewerben - Составить Anschreiben под вакансию
status - Мой баланс и статистика
jobs - Свежие вакансии
help - Справка
start - Начать заново
```

## Что нужно для активации в n8n

1. Telegram credential с токеном @BewerbZenBot
2. Claude API key в узле генерации
3. Вебхук зарегистрировать: GET `https://api.telegram.org/bot<TOKEN>/setWebhook?url=<N8N_WEBHOOK_URL>`
4. Активировать workflow `wSewHe5jlOvtPw1q` в n8n
